## ⚡ Features

* **Premium Developer Aesthetic:** Beautifully structured modern UI optimized for clear technical scannability.
* **Dynamic Theme Engine:** Native client-side toggle to switch instantly between light and dark modes (persists across sessions).
* **Interactive Visual Component Trees:** Responsive, vector-rendered SVG file structure diagrams replacing raw-text code trees.
* **Fluid Motion Hover States:** Elastic micro-interactions on side navigation bars, link trees, and status boxes.
* **Completely Frictionless:** Zero bulky third-party build systems, Node modules, or framework compilation pipelines—just pure, blazing-fast vanilla web components.
