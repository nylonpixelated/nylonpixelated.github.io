Server services

Services are the server-side singleton layer of the framework. They cleanly own runtime systems such as data, matchmaking, or game state orchestration.

--!strict
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local Feather = require(ReplicatedStorage.Feather)

export type DataServiceType = {
    Client: {},
    featherInit: (self: any) -> (),
    featherStart: (self: any) -> (),
}

local DataService = { Client = {} } :: DataServiceType

function DataService:featherInit()
    print("[DataService]: Instantiating structures...")
end

function DataService:featherStart()
    print("[DataService]: Service pipeline online.")
end

return Feather.CreateService(DataService)

Client controllers

Controllers isolate client-side execution and keep gameplay logic compartmentalized behind framework-safe entry points.

--!strict
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local Feather = require(ReplicatedStorage.Feather)

export type TestControllerType = {
    featherInit: (self: any) -> (),
    featherStart: (self: any) -> (),
}

local TestController = {} :: TestControllerType

function TestController:featherInit()
    print("[TestController]: Allocating buffers...")
end

function TestController:featherStart()
    print("[TestController]: Client logic active.")
end

return Feather.CreateController(TestController)

Lifecycle hooks

Feather follows a small and predictable lifecycle model so that initialization stays deterministic and easy to trace.

  • featherInit() is the configuration and setup phase.
  • featherStart() runs after modules are safely registered and ready to execute.

FeatherSpring

A precision interpolation package for smooth and responsive motion systems, especially useful for UI and physical effect pipelines.

local FeatherSpring = require(ReplicatedStorage.Packages.FeatherSpring)
local spring = FeatherSpring.new(1, 40, 4, 0)
spring:Update(deltaTime)

FeatherCleanup

Tracks and releases temporary connections and objects when a module or system is torn down, helping keep memory usage under control.

local FeatherCleanup = require(ReplicatedStorage.Packages.FeatherCleanup)
local Janitor = FeatherCleanup.new()
Janitor:Add(workspace.Part, "Destroy")
Janitor:Clean()

FeatherSignal

A lightweight event utility designed for fast dispatch and clear event-driven logic across modules.

local FeatherSignal = require(ReplicatedStorage.Packages.FeatherSignal)
local onAbilityUsed = FeatherSignal.new()

onAbilityUsed:Connect(function(abilityName)
    print("Casted skill:", abilityName)
end)

FeatherNet

An optimized network transport layer built to communicate cleanly between the server and connected clients with minimal overhead.

local FeatherNet = require(ReplicatedStorage.Packages.FeatherNet)
local ReliableRemote = FeatherNet.CreateEvent("ReliableRemote")
ReliableRemote:FireAllClients("MatchStart")
